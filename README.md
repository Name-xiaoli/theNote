# 这是个人本地使用的记录笔记功能的笔记本前端项目

# 使用的软件
vscoode, node.js, git

# 开发构建工具
Vite

# 前端
vue3渐进式框架(pinia vue-router axios), element-plus UI框架, wangEditor富文本编辑器

# 后端
json-server(利用其根据db.json文件快速实现增删改查功能)

# 使用方法
安装项目依赖
```sh
pnpm install
```

用于前端部署
```sh
pnpm dev
```

用于后端部署
```sh
pnpm jsondb
```
